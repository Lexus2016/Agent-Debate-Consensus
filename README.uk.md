## Навіщо Agent Debate Consensus?

Один AI дає вам одну точку зору. Agent Debate Consensus дає чотири — одночасно, у реальному часі, з аргументами один проти одного.

Різні моделі навчені на різних даних, мають різні упередження та різні стилі міркування. Змушуючи їх дебатувати на тему і захищати свої позиції перед опонентами, ви отримуєте повнішу картину, ніж здатна дати будь-яка окрема модель. Суперечності виходять на поверхню. Слабкі аргументи оскаржуються. Консенсус, коли він досягається, — зароблений.

Використовуйте платформу, щоб перевірити ідею на міцність, дослідити суперечливу тему з кількох кутів або спостерігати, як передові моделі підштовхують одна одну до точніших відповідей.

## Можливості

- **Багатоагентні дебати**: До 4+ AI-агентів (Kimi K2, Gemini 3 Pro, Claude Haiku 4.5, Grok 4.1 Fast) беруть участь у структурованих дебатах одночасно
- **Стримінг у реальному часі**: Потокова передача відповідей токен за токеном через Server-Sent Events — без очікування повної відповіді
- **Система @згадок**: Адресуйте питання конкретним агентам за допомогою `@Kimi`, `@Gemini`, `@Claude`, `@Grok`
- **Роль модератора**: Втручайтеся в будь-який момент, щоб перенаправити дебати, додати контекст або підштовхнути агентів до рішення
- **Розумний рушій відповідей**: Черга з пріоритетами та затримками не дозволяє агентам «говорити» одночасно
- **Каталог агентів**: Переглядайте та додавайте 400+ моделей із каталогу OpenRouter
- **Розширене мислення**: Повна підтримка моделей із можливостями міркування та chain-of-thought
- **Експорт у Markdown**: Завантажте всю дискусію у форматі `.md` для документації або передачі колегам
- **Копіювання повідомлень**: Скопіюйте будь-яке повідомлення в буфер обміну у форматі Markdown
- **Збереження сесій**: Розмови зберігаються в localStorage — продовжуйте з того місця, де зупинилися
- **Темна тема**: Сучасний темний інтерфейс преміум-класу, створений для тривалих сесій

## Як це працює

1. **Постановка теми** — Ви задаєте тему або питання. Усі активні агенти отримують її як контекст.
2. **Структуровані відповіді** — Кожен агент формулює незалежну позицію, не знаючи, що готують інші.
3. **Перехресний обмін** — У наступних раундах кожен агент отримує попередні відповіді інших агентів, що забезпечує справжню реакцію та контраргументи.
4. **Модерація** — Ви втручаєтеся в будь-який момент: уточнюєте терміни, оскаржуєте слабкий аргумент або просите конкретного агента захистити свою позицію.
5. **Зближення позицій** — Через ітеративний обмін агенти уточнюють позиції, визнають сильніші аргументи та рухаються до консенсусу — або виявляють непримиренні розбіжності, про які варто знати.

## Встановлення

**1. Клонування репозиторію**

```bash
git clone https://github.com/Lexus2016/Agent-Debate-Consensus.git
cd Agent-Debate-Consensus
```

**2. Встановлення залежностей**

```bash
npm install
```

**3. Налаштування середовища**

Створіть файл `.env` у корені проекту:

```env
OPENROUTER_API_KEY=your_key_here
```

Отримайте API-ключ на [openrouter.ai/keys](https://openrouter.ai/keys).

**4. Запуск сервера розробки**

```bash
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000).

## Використання

1. Оберіть AI-агентів на бічній панелі
2. Введіть тему або питання, щоб розпочати дебати
3. Спостерігайте, як агенти обговорюють, сперечаються та розвивають думки один одного
4. Використовуйте `@згадки`, щоб поставити питання конкретному агенту
5. Втручайтеся в будь-який момент, щоб спрямувати обговорення або додати свою точку зору
6. Агенти працюють до консенсусу через структурований діалог
